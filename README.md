# Full-Stack Web Application

A modern full-stack web application built with Node.js/Express backend and Next.js frontend, following industry best practices for security, performance, and maintainability.

## Features

- Authentication with JWT (Access + Refresh tokens)
- Role-based access control
- Input validation and sanitization
- API rate limiting
- Comprehensive logging
- Responsive UI with Tailwind CSS
- PWA capabilities
- TypeScript support
- Comprehensive test coverage
- CI/CD pipeline with GitHub Actions
- Docker containerization

## Tech Stack

### Backend
- Node.js 18
- Express.js
- PostgreSQL with Knex.js
- JWT authentication
- Joi validation
- Winston logging
- Jest testing

### Frontend
- Next.js 13
- React
- TypeScript
- Tailwind CSS
- Axios
- React Testing Library

### DevOps
- Docker
- GitHub Actions
- PostgreSQL
- pgAdmin
